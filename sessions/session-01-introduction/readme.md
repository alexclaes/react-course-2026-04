# Session 01: Einführung in React

## 💪 Motivation

Diese Session führt dich durch die Grundlagen der JavaScript-Bibliothek React.

## 🎯 Lernziele

- Grundlegende Konzepte von React kennen
- React Components und Komposition verstehen
- Unterschiede zwischen JSX und HTML kennen
- Styles auf React Components anwenden
- Globale Styles auf eine React-Anwendung anwenden
- Die Chrome-Erweiterung "React Developer Tools" nutzen
